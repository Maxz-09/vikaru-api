Vikaru-API

Vikaru-API is a Web Server-based API that can be integrated into AutoResponder applications.

Information API

Select an API to see more information.

How to use the API

  1. Add a new AutoResponder rule.

  2. In your AutoResponder rule, adjust the keyword command to your needs.

  3. Select the Web server option in your AutoResponder rule and enter the API URL.

  4. The Basic Auth and Headers options are optional. Please refer to the comments in the code above if you'd like to use them.

  5. That's it! You are now ready to test your web server integration.
